1. Пункт проработан.
    При добавлении существующего номера - this number already registered. Добавлять новые номера можно. Добавлять новые данные на зарегестрированный номер нельзя. Один номер - одни данные.
    Можно на одно имя несколько разных номеров и тд.

2. Пункт:
    При поиске зареганого номера выдает инфу по персоне. При поиске не зареганой инфы - дает уведомление с коткретным указанием, что нет данных. Также и с номером

3 Пункт:
    При попытке модифицировать данные номера, которого нет в базе - дает уведомление, что конкретного номера нет
    При попытке ввести в качестве номера не числа - выдает уведомление об этом. ( Нет проверки на правильность номера) В целом моно из пред задания домашнего сделать функцию и тоже вставить.
    Разбил возможность изменение на два параметра: регион и фулл нейм. Имя и фамилию объединил для того чтобы автоматически меняло в словаре значение Full name.
    В этом случае, если нужно менять только имя, то нужно будет подтвердить фамилию, чтобы не было офибки с Full name
    Номер телефона изменять нельзя
4 Пункт:
    Удаляет всю информацию, связанную с введенным номером
    При попытке удалить данные несуществующего номера выдает уведомление.
