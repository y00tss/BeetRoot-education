1. Пункт проработан.
    При добавлении существующего номера выводится сообщение "This number is already registered".
    Можно добавлять новые номера. Но нельзя добавлять новые данные для уже зарегистрированного номера.
    Каждому номеру соответствуют только определенные данные. Также можно иметь несколько разных номеров на одно имя и т.д.

2. Пункт:
    При поиске зарегистрированного номера выводится информация о соответствующем лице.
    При поиске незарегистрированного номера выводится уведомление с конкретным указанием, что данных нет.
    То же самое относится и к номеру.

3 Пункт:
    При попытке изменить данные номера, которого нет в базе, выводится уведомление, что такого номера не существует.
    При попытке ввести нечисловое значение в качестве номера выводится соответствующее уведомление.
    Однако, не выполняется проверка на правильность номера. В целом, можно взять задание домашнего задания и создать функцию для этого.
    Я разделил возможность изменения на два параметра: регион и полное имя (Full name). И
    мя и фамилию объединил, чтобы автоматически менять значение полного имени в словаре.
    В этом случае, если нужно изменить только имя, будет требоваться подтверждение фамилии, чтобы избежать ошибок в полном имени.
    Номер телефона изменять нельзя.
4 Пункт:
    Удаляет всю информацию, связанную с введенным номером.
    При попытке удалить данные для несуществующего номера выводится уведомление.
5 Пункт:
    Закрывает приложение без ошибок))))
